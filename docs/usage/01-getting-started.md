# 快速开始

> 30 秒上手 obsidian-image-referer

## 前置条件

- 已安装 Obsidian 桌面客户端（Windows / macOS / Linux）
- Obsidian 版本 ≥ 0.15.0

## 快速配置

1. 安装并启用插件（详见 [安装指南](./02-installation.md)）
2. 打开 **Settings → Community plugins → Image Referer**
3. 在 **Referer** 输入框中填入目标 Referer 值（例如 `https://mp.weixin.qq.com`）
4. 返回笔记，之前无法显示的图片现在应该能正常加载了

## 效果验证

配置完成后，打开一篇包含需要 Referer 的远程图片的笔记，图片应能正常显示而非显示为空白或 403 错误图。
